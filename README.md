# Show Up. Cash Out. Reign. Web Edition

This repository hosts the public static web edition of the ebook.

## Reader Access

- Public site entry point: `index.html`
- PDF download is bundled in `assets/`
- Reader password: shared separately by the publisher

## Notes

- The site is static and GitHub Pages friendly.
- The password gate is client-side, so it is suitable for light access control rather than hard document security.
